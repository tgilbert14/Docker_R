**--> steps I did for app example**-->
create dockerFile w/no extention:
----------------------------------------------- *Example*
# Base R Shiny image
FROM rocker/shiny

# Make a directory in the container
RUN mkdir /home/shiny-app

# Install R dependencies
RUN R -e "install.packages(c('tidyverse', 'ggplot2', 'gapminder'))"

# Copy the Shiny app code
COPY app.R /home/shiny-app/app.R

# Expose the application port
EXPOSE 8180

# Run the R Shiny app
CMD Rscript /home/shiny-app/app.R
-----------------------------------------------

Set dir-> 'cd ..' to move back directories, then set to directory folder/repo -> 

cd Docker

Then run to get image on docker hub (build/login/push) -> 

docker build -t tsgilbert/tims-docker-app2:V1 -f Dockerfile .
docker login -u "tsgilbert" -p "ShawnTim14!" docker.io
docker push tsgilbert/tims-docker-app2:V1
